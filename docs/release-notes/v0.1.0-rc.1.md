# Codex Auth Bar v0.1.0-rc.1

The first release candidate of the native macOS menu-bar app for managing
Codex accounts, usage limits, and CLI profiles.

## Highlights

- Switch accounts from the menu bar, including a graceful **Switch & Restart**
  flow for Codex Desktop.
- Small, medium, and large WidgetKit widgets show 5-hour and weekly limits for
  managed accounts.
- Import and export standard Codex auth, JSON arrays, folders, and CLIProxyAPI
  records.
- Browser, device-code, and API-key login run in an isolated temporary
  `CODEX_HOME`.
- Atomic auth replacement, private file permissions, backups, transaction
  journaling, and crash recovery.
- Independent Codex CLI profile discovery and launch.
- English and Russian localizations on Apple Silicon and Intel Macs.

## Requirements

- macOS 14 or newer.
- Codex CLI for login and profile-launch workflows.

## Distribution

The downloadable DMG is published only after the protected release workflow
builds a universal app, signs it with Developer ID, notarizes it with Apple,
staples the notarization ticket, and verifies the final image. A SHA-256 file is
published next to the DMG.

Unsigned CI artifacts are development builds and are not release assets.

## Security and compatibility

Codex Auth Bar stores managed auth snapshots locally because Codex itself uses
plaintext `auth.json`. Read the [security model](../security.md) before use.
Remote usage refresh uses unofficial ChatGPT endpoints and can be disabled.

Behavioral compatibility is pinned to
[`Loongphy/codex-auth@22d87d1`](https://github.com/Loongphy/codex-auth/tree/22d87d1531420102fa2f3d51d134f29344dda27c).
Codex Auth Bar is an independent project and is not affiliated with OpenAI or
Loongphy.
