# Codex Auth Bar v0.1.0-alpha.1

This is the first installable developer preview of Codex Auth Bar.

## Important distribution notice

The attached app is ad-hoc signed. It is **not Developer ID signed or notarized
by Apple**. macOS can block the first launch; after attempting to open the app,
you may need to approve it through System Settings → Privacy & Security → Open
Anyway. Do not install this preview if you require a normally trusted macOS
distribution.

Both the DMG and ZIP are universal (`arm64 + x86_64`). SHA-256 files are
attached for integrity checking.

## Highlights

- Menu-bar switching between managed Codex accounts.
- Graceful **Switch & Restart** for Codex Desktop.
- Small, medium, and large widgets showing 5-hour and weekly limits.
- Account aliases, previous-account switching, import/export, backups, and
  recovery.
- Isolated browser, device-code, and API-key login workflows.
- Independent named Codex CLI configuration profiles.
- English and Russian localizations.

## Privacy

Codex credentials remain in local Codex auth files; the app has no analytics or
external backend. Remote usage refresh uses unofficial ChatGPT endpoints and
can be disabled. Widget snapshots contain no credentials, email addresses, or
Codex account identifiers.

Read the complete [security model](../security.md) before use.
