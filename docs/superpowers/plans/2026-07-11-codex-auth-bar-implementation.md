# Codex Auth Bar implementation plan

The accepted plan is implemented as these independently verified slices:

1. OSS/Xcode/Swift package scaffold and build loop.
2. Auth models, parsing, schema migration, and profile validation.
3. Secure registry storage, concurrency detection, backups, and recovery.
4. Account switch/import/export/remove/alias/clean workflows.
5. Usage/local fallback, profiles, auto-switch, and codext verification.
6. SwiftUI menu bar, management window, Settings, and Codex process control.
7. CI, signing/notarization scripts, documentation, and release validation.
